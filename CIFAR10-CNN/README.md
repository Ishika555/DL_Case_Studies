# CIFAR-10 Image Classification using CNN

This project implements a Convolutional Neural Network (CNN) using TensorFlow to classify images from the CIFAR-10 dataset.

## Features
- Uses built-in CIFAR-10 dataset
- CNN model with multiple convolution layers
- Accuracy visualization
- Training and testing included

## Technologies
- Python
- TensorFlow / Keras
- Matplotlib

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run:
   python cifar10_cnn.py
